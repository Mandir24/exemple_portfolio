## 🌐 Aperçu du Projet

Ce dépôt contient le code source de mon portfolio personnel. Il a pour objectif de présenter mon parcours académique, mes compétences techniques et mes réalisations concrètes dans le domaine de la **Data Science**, avec un accent particulier sur la **Visualisation** et l'**Ingénierie des Données**.

Mon but est de transformer des données brutes en informations stratégiques, accessibles et visuelles.

## 🚀 Sections Clés

Le portfolio est structuré pour offrir une vue complète de mon profil :

* **Profil** (`#description`) : Ma présentation, mon parcours (BUT Sciences des Données - IUT Lisieux) et ma recherche de stage (dès **mars**).
* **Compétences** (`#compétences`) : Détail de mes expertises en **Analyse de Données**, **Ingénierie des Données**, **Statistiques** et **Machine Learning/NLP**.
* **Outils et Technologies** (`#outils`) : Les outils que je maîtrise, incluant Python, SQL, Power BI, Tableau, QGIS, et divers frameworks (Pandas, scikit-learn).
* **Expériences** (`#expériences`) : Mes expériences professionnelles, y compris mon futur stage de **Data Engineer** à l'Agence de l'eau Seine Normandie & Cater COM.
* **Projets** (`#projets`) : Une galerie interactive de mes projets, couvrant l'analyse de séries temporelles, le datamining (kNN), et des dashboards économiques et financiers (EcoDashboard, BourseTrack).
* **Poster** (`#poster`) : Les posters académiques réalisés durant mon BUT.

## 📂 Structure du Dépôt

Le site utilise un modèle basé sur [Agency Theme de Start Bootstrap](https://startbootstrap.com/theme/agency). Les fichiers principaux sont :

* `index.html` : La page d'accueil principale du portfolio.
* `css/agency.css` : Styles personnalisés (CSS).
* `js/` : Scripts JavaScript pour les animations et le comportement interactif.
* `projet/` : Dossier contenant les pages dédiées à chaque projet.
* `img/` : Dossier des assets (logos, images, icônes).
